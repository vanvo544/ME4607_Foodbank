Khi muốn push code lên github thì chỉ cần chạy file github.bat và truyền thêm message vào sau là được

Ví dụ:

```bash
.\github.bat "add something"
```

Nếu muốn gõ từ từ thì

Step 1: add những file cần push lên github
```bash
git add .
```
-> add tất cả

```bash
git add public
```
-> add mỗi folder public

Step 2: Thêm message khi push lên để lần sau có thể xem lại mình push vì lí do, tại sao push
```bash
git commit -m "add something feature"
```

Step 3: Đẩy lên github
```bash
git push origin main
```
đôi khi mặc định nó ko là branch main thì là master, khi đó
```bash
git push origin master
```
Xem ở đây để biết branch nào
